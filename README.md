# CounterMVVM

Counter is a simple Android application built using Jetpack Compose and follows the MVVM (Model-View-ViewModel) architecture pattern. It demonstrates modern Android development practices, including declarative UI and effective state management.


After setting up the environment clone this project to your desired location.

```bash
git clone git@github.com:253nomi/android_counterapp_mvvm.git
```
